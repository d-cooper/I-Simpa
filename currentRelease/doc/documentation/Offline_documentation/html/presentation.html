GUI Presentation¶

The graphic user interface (GUI) is composed by the following components:

  • a menu bar (‘File’, ‘Edition’, ‘Simulation’, ‘View’, ‘Windows’, ‘Help’) on the main window

  • several independent ‘Toolbars’ (dockable/undockable)

  • a ‘Project’ window (dockable/undockable), with three tabs ‘Scene’, ‘Calculation’, ‘Results” (tabs can be re-ordered within the windows)

  • a ‘Properties’ window (dockable/undockable)

  • a ‘Console’ window (dockable/undockable) with two tabs ‘Message’ and ‘Python’ (tabs can be re-ordered within the windows)

  • a ‘Main’ window with the 3D view

I-Simpa user interface.¶

Windows properties:

  • User can rearrange and resize all components in the computer screen by selecting and moving the corresponding component.

  • You can come back to the default view by selecting the option ‘Reinitialize interface’ in the ‘Windows’ menu.
